===============================================================================
                              项目概述
===============================================================================

这是一个基于AI的Android应用，可以通过上传视频来分析和计数俯卧撑动作。
应用使用Google ML Kit的姿态检测技术，结合自定义的计数算法，实现了：
- 智能识别俯卧撑动作
- 实时计数
- 姿态规范检测
- 视频播放与分析同步

===============================================================================
                              核心功能
===============================================================================

1. ✅ 视频上传和选择
2. ✅ AI姿态检测（Google ML Kit）
3. ✅ 俯卧撑智能计数
4. ✅ 实时视频播放（ExoPlayer）
5. ✅ 动作状态反馈
6. ✅ 姿态规范检测
7. ✅ Material Design UI

===============================================================================
                              技术架构
===============================================================================

核心技术：
- Kotlin：主开发语言
- Google ML Kit：姿态检测
- ExoPlayer (Media3)：视频播放
- CameraX：相机支持（预留）
- Kotlin Coroutines：异步处理
- Material Components：UI设计

主要组件：
- MainActivity：主界面，视频选择
- VideoAnalysisActivity：视频分析和播放
- PushUpDetector：检测算法核心
- PoseOverlayView：姿态可视化

===============================================================================
                              快速开始
===============================================================================

1. 克隆项目：
   git clone <repository-url>
   cd PushUpCounter

2. 设置环境（可选）：
   ./setup.sh

3. 使用Android Studio打开项目

4. 同步Gradle依赖

5. 运行应用：
   ./gradlew installDebug
